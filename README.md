Live at: [FutureSelf.joinhustleclubcom](https://futureself.joinhustleclub.com)

---

## What Is FutureSelf?

FutureSelf is a Chrome extension that blocks distracting websites during your evening and sleep hours — including work tools, AI tools, and search engines. Not just social media.

The core insight: **fake productivity is a sleep killer too.** Editing a doc at midnight, running a ChatGPT rabbit hole at 1 AM, reorganizing Notion "planning for tomorrow" — these feel productive but they're not. They're just avoiding the discomfort of closing the laptop.

FutureSelf blocks all of it. Then asks you one question before letting you through.

> *"Tomorrow will thank you."*

## This Repo

This is the marketing landing page for FutureSelf. It is a static site built with HTML, CSS, and vanilla JS.

### Structure

```
futureself-landing/
├── index.html          # Main landing page
├── style.css           # All styles
├── script.js           # Interactions and animations
└── auth-callback.html  # Auth redirect handler
```

### Running Locally

```bash
# Just open index.html in your browser
open index.html

# Or serve with any static server
npx serve .
```
